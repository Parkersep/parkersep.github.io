ESP32-S3 Teleoperation Dev Board

Personal Project · In Progress

An ESP32-S3 development board designed in KiCad — USB-C with onboard voltage regulation, RST/BOOT buttons, an addressable RGB LED, and a broken-out GPIO / display header, built around the ESP32-S3-WROOM module.

I'm actively expanding it with peripherals for different robotic teleoperation scenarios — a reusable base board for driving leader-arm inputs, buttons, and sensors.

Mecanum Robot & Wireless Controller

Personal Project · Fall 2025

A ground-up mecanum robot with a matching handheld wireless controller — both built on electrical systems I designed. The controller carries an XBee radio, an IMU, and a potentiometer, with buttons for tilt-drive control, zero-turn, and communication; C++ firmware on an Arduino Mega implements mecanum-wheel kinematics, motor control, and ultrasonic sensing for obstacle avoidance.

Teleoperation & Robot Foundation Models

Sensori Robotics · 2026

I built the teleoperation pipeline in Isaac Sim for collecting manipulation demonstrations with the stock OpenArm, and use it to fine-tune and evaluate robot foundation models.
